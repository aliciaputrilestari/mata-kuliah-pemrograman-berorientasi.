02 - NAMED CONSTRUCTOR (CLASS SEPEDA)

Setelah class didefinisikan, kita dapat membuat object atau instance dari class tersebut di dalam fungsi main(). Proses ini disebut instansiasi, di mana objek dialokasikan ke dalam memori.

// Nama  : Alicia Putri Lestari
// NIM   : 251410011
// Kelas : SI2A

class Sepeda {
  String? merk;
  int? gear;

  Sepeda(this.merk, this.gear);

  Sepeda.standard() {
    merk = "Polygon";
    gear = 21;
  }
}

void main() {
  var sepeda1 = Sepeda.standard();

  print("Merk Sepeda: ${sepeda1.merk}");
  print("Jumlah Gear: ${sepeda1.gear}");
}

03 - CONSTANT CONSTRUCTOR (CLASS UKURAN)

Constructor adalah metode inisialisasi yang dipanggil secara otomatis saat objek baru diciptakan dalam memori sistem. Di Dart, kita bisa menggunakan formal parameters (this.attribute) untuk menyederhanakan proses pengisian nilai attribute.

// Nama  : Alicia Putri Lestari
// NIM   : 251410011
// Kelas : SI2A

class Ukuran {
  final int panjang;
  final int lebar;

  const Ukuran(this.panjang, this.lebar);
}

void main() {
  const ukuranMeja = Ukuran(120, 60);

  print("Ukuran: ${ukuranMeja.panjang} x ${ukuranMeja.lebar}");
}

05 - GETTER (CLASS FILM)

Encapsulation adalah teknik untuk menyembunyikan detail implementasi internal. Di Dart, kita menggunakan awalan underscore (_) untuk membuat attribute bersifat private, sehingga hanya bisa diakses melalui getter dan setter.

// Nama  : Alicia Putri Lestari
// NIM   : 251410011
// Kelas : SI2A

class Film {
  String _judul;

  Film(this._judul);

  String get judul => _judul;
}

void main() {
  var film1 = Film("Avengers");

  print("Judul Film: ${film1.judul}");
}

07 - ENKAPSULASI CLASS REKENING


Latihan lanjutan menggunakan Named Parameters dan operator required. Teknik ini membuat pembuatan
objek menjadi lebih deskriptif dan aman karena compiler akan memberikan peringatan jika data 
wajib tidak diisi.




// Nama  : Alicia Putri Lestari
// NIM   : 251410011
// Kelas : SI2A

class Rekening {
  String _namaPemilik = "";
  double _saldo = 0;

  String get namaPemilik => _namaPemilik;
  double get saldo => _saldo;

  set namaPemilik(String value) {
    _namaPemilik = value;
  }

  void tambahSaldo(double jumlah) {
    if (jumlah > 0) {
      _saldo += jumlah;
    }
  }
}

void main() {
  var rekening1 = Rekening();

  rekening1.namaPemilik = "Alicia";
  rekening1.tambahSaldo(500000);

  print("Pemilik: ${rekening1.namaPemilik}");
  print("Saldo: ${rekening1.saldo}");
}












08 - SISTEM LOGIN VERSI LAIN

// Nama  : Alicia Putri Lestari

class SistemLogin {
  String _username = "user1";
  String _password = "12345678";

  bool cekLogin(String user, String pass) {
    return user == _username && pass == _password;
  }
}

void main() {
  var login = SistemLogin();

  if (login.cekLogin("user1", "12345678")) {
    print("Akses diterima");
  } else {
    print("Akses ditolak");
  }
}
