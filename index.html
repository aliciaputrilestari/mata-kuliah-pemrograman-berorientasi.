01 - DEFAULT CONSTRUCTOR (CLASS HEWAN)

Constructor adalah fungsi khusus di dalam class yang digunakan untuk memberikan nilai awal pada objek saat objek dibuat. Constructor akan otomatis dipanggil ketika kita membuat objek dari class tersebut. Dengan constructor, kita bisa langsung mengisi nilai atribut tanpa harus mengubahnya secara manual setelah objek dibuat. 

// Nama  : Alicia Putri Lestari
// NIM   : 251410011
// Kelas : SI2A

class Hewan {
  String nama;
  String jenis;

  Hewan(this.nama, this.jenis);
}

void main() {
  var hewan1 = Hewan("Kucing", "Mamalia");

  print("Nama Hewan: ${hewan1.nama}");
  print("Jenis: ${hewan1.jenis}");
}

02 - NAMED CONSTRUCTOR (CLASS SEPEDA)

Named constructor adalah constructor yang memiliki nama tambahan sehingga satu class dapat memiliki lebih dari satu constructor. Hal ini berguna ketika kita ingin membuat objek dengan cara yang berbeda atau dengan nilai awal yang berbeda.

// Nama  : Alicia Putri Lestari
// NIM   : 251410011
// Kelas : SI2A

class Sepeda {
  String? merk;
  int? gear;

  Sepeda(this.merk, this.gear);

  Sepeda.standard() {
    merk = "Polygon";
    gear = 21;
  }
}

void main() {
  var sepeda1 = Sepeda.standard();

  print("Merk Sepeda: ${sepeda1.merk}");
  print("Jumlah Gear: ${sepeda1.gear}");
}

03 - CONSTANT CONSTRUCTOR (CLASS UKURAN)

Constant constructor digunakan untuk membuat objek yang nilainya tidak dapat diubah (immutable). Constructor ini biasanya digunakan pada class yang memiliki atribut final sehingga nilainya tetap setelah objek dibuat.

// Nama  : Alicia Putri Lestari
// NIM   : 251410011
// Kelas : SI2A

class Ukuran {
  final int panjang;
  final int lebar;

  const Ukuran(this.panjang, this.lebar);
}

void main() {
  const ukuranMeja = Ukuran(120, 60);

  print("Ukuran: ${ukuranMeja.panjang} x ${ukuranMeja.lebar}");
}

04 - INITIALIZER LIST (CLASS SEGITIGA)

Initializer list adalah cara untuk menginisialisasi nilai variabel sebelum constructor dijalankan. Biasanya digunakan ketika variabel bersifat final atau ketika kita ingin melakukan proses tertentu sebelum constructor selesai. 

// Nama  : Alicia Putri Lestari
// NIM   : 251410011
// Kelas : SI2A

class Segitiga {
  double alas;
  double tinggi;
  double luas;

  Segitiga(this.alas, this.tinggi)
      : luas = 0.5 * alas * tinggi;
}

void main() {
  var segitiga1 = Segitiga(8, 6);

  print("Luas Segitiga: ${segitiga1.luas}");
}

05 - GETTER (CLASS FILM)

Getter adalah metode yang digunakan untuk mengambil atau membaca nilai dari variabel private. Dengan getter, data dalam class tetap terlindungi tetapi masih bisa diakses secara aman dari luar class.

// Nama  : Alicia Putri Lestari
// NIM   : 251410011
// Kelas : SI2A

class Film {
  String _judul;

  Film(this._judul);

  String get judul => _judul;
}

void main() {
  var film1 = Film("Avengers");

  print("Judul Film: ${film1.judul}");
}

06 - SETTER (CLASS FILM)

Setter adalah metode yang digunakan untuk mengubah nilai variabel private dengan aturan tertentu. Biasanya setter digunakan untuk melakukan validasi data sebelum nilai tersebut disimpan ke dalam variabel.

07 - ENKAPSULASI CLASS REKENING


Encapsulation adalah konsep dalam Object Oriented Programming (OOP) yang digunakan untuk melindungi
data dalam class agar tidak bisa diakses langsung dari luar.
Data biasanya dibuat private dan hanya dapat diakses melalui getter dan setter.





// Nama  : Alicia Putri Lestari
// NIM   : 251410011
// Kelas : SI2A

class Rekening {
  String _namaPemilik = "";
  double _saldo = 0;

  String get namaPemilik => _namaPemilik;
  double get saldo => _saldo;

  set namaPemilik(String value) {
    _namaPemilik = value;
  }

  void tambahSaldo(double jumlah) {
    if (jumlah > 0) {
      _saldo += jumlah;
    }
  }
}

void main() {
  var rekening1 = Rekening();

  rekening1.namaPemilik = "Alicia";
  rekening1.tambahSaldo(500000);

  print("Pemilik: ${rekening1.namaPemilik}");
  print("Saldo: ${rekening1.saldo}");
}












08 - SISTEM LOGIN VERSI LAIN


Contoh ini menunjukkan bagaimana konsep enkapsulasi digunakan dalam sistem login.
Username dan password disimpan dalam class, kemudian proses login dilakukan dengan cara membandingkan
data yang dimasukkan pengguna dengan data yang ada di dalam class.





// Nama  : Alicia Putri Lestari
// NIM   : 251410011
// Kelas : SI2A

class SistemLogin {
  String _username = "user1";
  String _password = "12345678";

  bool cekLogin(String user, String pass) {
    return user == _username && pass == _password;
  }
}

void main() {
  var login = SistemLogin();

  if (login.cekLogin("user1", "12345678")) {
    print("Akses diterima");
  } else {
    print("Akses ditolak");
  }
}
