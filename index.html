03 - NAMED CONSTRUCTOR - CONTOH DETAIL

Dalam bahasa Dart, kamu bisa membuat lebih dari satu konstruktor di dalam satu kelas dengan menggunakan Named Constructor. Fitur ini sangat berguna karena memberikan cara alternatif untuk menginisialisasi objek, misalnya saat mengambil data dari format JSON.

// Nama   : Alicia Putri Lestari
// Nim    : 251410011
// Kelas  : SI2A

import 'dart:convert';

class Mobil {
  String? merk;
  int? tahun;

  // Parameterized constructor
  Mobil(this.merk, this.tahun);

  // Named constructor
  Mobil.baru() {
    merk = 'BMW';
    tahun = 2020;
  }

  // Named constructor dengan parameter
  Mobil.dariJson(Map json) {
    merk = json['merk'];
    tahun = json['tahun'];
  }

  // Override toString for better print output
  @override
  String toString() {
    return 'Mobil{merk: $merk, tahun: $tahun}';
  }
}

void main() {
  print('--- Demonstrasi Class Mobil ---');

  // 1. Menggunakan constructor default (parameterized constructor)
  var mobilA = Mobil('Honda', 2020);
  print('Mobil A (constructor default): $mobilA');

  // 2. Menggunakan named constructor Mobil.baru()
  var mobilB = Mobil.baru();
  print('Mobil B (named constructor .baru): $mobilB');

  // 3. Menggunakan named constructor Mobil.dariJson()
  String jsonString = '{"merk": "Suzuki", "tahun": 2021}';
  Map jsonData = jsonDecode(jsonString);
  var mobilC = Mobil.dariJson(jsonData);
  print('Mobil C (named constructor .dariJson): $mobilC');

  print('\n--- Akhir Demonstrasi ---');
}

04 - CONSTANT CONSTRUCTOR

Konstruktor konstan berfungsi untuk membuat objek yang bersifat immutable atau tidak dapat diubah setelah dibuat. Keunggulannya, jika kamu membuat objek dengan kata kunci  const  dan menggunakan nilai yang sama, Dart akan menggunakan instance yang sama di memori. Hal ini membuat penggunaan sumber daya menjadi lebih efisien.

// Nama   : Alicia Putri Lestari
// Nim    : 251410011
// Kelas  : SI2A

class Warna {
  final int r, g, b;
  const Warna(this.r, this.g, this.b);

  @override
  String toString() {
    return 'Warna(r: $r, g: $g, b: $b)';
  }
}

void main() {
  const merah = Warna(255, 0, 0);
  const merahLagi = Warna(255, 0, 0); // Menggunakan instance yang sama di memory

  print(merah);
  print(merahLagi);
  print('Apakah identik? ${identical(merah, merahLagi)}');
}

05 - CONSTRUCTOR DENGAN INITIALIZER LIST

Initializer List berfungsi untuk menginisialisasi properti (field) kelas sebelum badan konstruktor dieksekusi. Fitur ini sering digunakan untuk melakukan validasi data atau mengubah nilai sebelum dimasukkan ke dalam field yang bersifat  final .

// Nama   : Alicia Putri Lestari
// Nim    : 251410011
// Kelas  : SI2A

class Titik {
  final double x, y;

  Titik(double x, double y)
    : x = x.abs(),
      y = y.abs() {
    print('Titik dibuat di (${this.x}, ${this.y})');
  }
}

void main(){
  print('Membuat objek Titik pertama:');
  var titik1 = Titik(3.0, 4.0);
  print('Titik 1: x = ${titik1.x}, y = ${titik1.y}');

  print('\nMembuat objek Titik kedua dengan nilai negatif:');
  var titik2 = Titik(-5.0, 10.0);
  print('Titik 2: x = ${titik2.x}, y = ${titik2.y}');
}

06 - PRAKTIK 1 - MULTIPLE CONSTRUCTOR

Latihan membuat kelas Mahasiswa dengan tiga jenis konstruktor yang berbeda. Tujuannya adalah agar pembuatan objek menjadi lebih fleksibel, bisa disesuaikan dengan jenis data yang tersedia:

// Nama   : Alicia Putri Lestari
// Nim    : 251410011
// Kelas  : SI2A

class Mahasiswa {
  String? nama;
  String? nim;

  Mahasiswa(this.nama, this.nim);

  Mahasiswa.dariMap(Map data) {
    nama = data['nama'];
    nim = data['nim'];
  }

  Mahasiswa.tanpaNama() {
    nama = 'No Name';
    nim = '000000';
  }

  void info() => print('Mahasiswa: $nama, NIM: $nim');
}

void main() {
  var m1 = Mahasiswa(''Cia, '251410010');
  var m2 = Mahasiswa.dariMap({'nama': 'Alicia', 'nim': '251410011'});
  var m3 = Mahasiswa.tanpaNama();

  m1.info();
  m2.info();
  m3.info();
}










07 - PENGANTAR ENKAPSULASI


Konsep: Menyembunyikan detail cara kerja program dari luar, supaya hanya bagian yang perlu yang bisa diakses.

 

Tujuan:

 

- Melindungi data agar tidak diubah sembarangan

​

- Mengatur siapa yang bisa mengakses apa

​

- Memudahkan perubahan kode tanpa gangguan ke bagian lain

 

Cara di Dart:

 

- Tambah  _  di depan nama untuk bikin private (hanya bisa dipakai di file yang sama)

​

- Public (bawaan): Bisa diakses dari mana saja




// Nama   : Alicia Putri Lestari
// Nim    : 251410011
// Kelas  : SI2A

class Person {
  String _name; // Private: Hanya bisa diakses di file ini
  int age;      // Public: Dapat diakses bebas

  Person(this._name, this.age);

  void info() => print('Nama: $_name, Umur: $age');
}

void main() {
  var p = Person('Alicia Putri Lestari', 18);
  p.info();
}












08 - GETTER & SETTER


Getter: Memungkinkan membaca data private dengan kontrol.

 

- Sintaks 1 (Arrow):  tipeData get namaGetter => _field; 

​

- Sintaks 2 (Block):  tipeData namaGetter() { return _field; } 

 

Setter: Memungkinkan mengubah data dengan validasi.

 

- Sintaks 1 (Set Keyword):  set namaSetter(value) { ... } 

​

- Sintaks 2 (Method Approach):  void namaSetter(value) { ... }




// Nama   : Alicia Putri Lestari
// Nim    : 251410011
// Kelas  : SI2A

class Person {
  String _name;
  int age;
  Person(this._name, this.age);
  
  set setName1(String name) {
    _name = name;
  }

  void setName(String name) {
    _name = name;
  }

  String getName() {
    return _name;
  }
}

void main() {
  Person person = Person("Alicia", 18);
  print('Umur: ${person.age}');
  
  person.setName('Afrizal');
  print('Nama (via Method): ${person.getName()}');
  
  person.setName1 = 'Alicia Putri Letari';
  print('Nama (via Setter): ${person.getName()}');
}

Meningkatkan integritas data dengan validasi nilai pada kelas Produk sebelum disimpan:

​

- Harga produk harus lebih besar dari 0




// Nama   : Alicia Putri Lestari
// Nim    : 251410011
// Kelas  : SI2A

class Produk {
  String _nama;
  double _harga;
  
  Produk(this._nama, this._harga);
  
  String get nama => _nama;
  double get harga => _harga;
  
  set nama(String value) {
    if (value.length >= 3) {
      _nama = value;
    } else {
      print('Gagal memperbarui nama: Nama harus memiliki minimal 3 karakter.');
    }
  }
  
  set harga(double value) {
    if (value > 0) {
      _harga = value;
    } else {
      print('Gagal memperbarui harga: Harga harus lebih besar dari 0.');
    }
  }
}

void main() {
  var produk1 = Produk('Leptop Lenovo', 3500.0);

  print('--- Informasi Produk Awal ---');
  print('Nama Produk: ${produk1.nama}');
  print('Harga Produk: Rp ${produk1.harga}');

  print('\n--- Memperbarui Produk ---');
  produk1.nama = 'Mouse Gaming';
  produk1.harga = 1200.0;
  print('Nama (setelah update): ${produk1.nama}');

  print('\n--- Menguji Validasi Setter ---');
  produk1.nama = 'TV'; // Gagal
  produk1.harga = -100.0; // Gagal
}

class User {
  String _username;
  String _password;
  String _email;

  User(this._username, this._password, this._email);

  // Getter untuk semua field
  String get username => _username;
  String get email => _email;

  // Setter dengan validasi
  set username(String v) {
    if (v.length >= 5) {
      _username = v;
    } else {
      print('Username min 5 karakter');
    }
  }

  set password(String v) {
    if (v.length >= 8) {
      _password = v;
    } else {
      print('Password min 8 karakter');
    }
  }

  set email(String v) {
    if (v.contains('@')) {
      _email = v;
    } else {
      print('Format email salah (Harus @)');
    }
  }

  void info() => print('User: $_username, Pass: $_password, Email: $_email');
}

void main() {

  user.username = 'aliciaputrilestari';

11 - STUDI KASUS - SISTEM LOGIN


 Logika autentikasi dan prosedur perubahan password




// Nama   : Alicia Putri Lestari
// Nim    : 251410011
// Kelas  : SI2A

class User {
  String _username;
  String _password;

  User(this._username, this._password);
  String get username => _username;

  bool login(String username, String password) {
    return _username == username && _password == password;
  }

  void gantiPassword(String lama, String baru) {
    if (_password == lama) {
      if (baru.length >= 8) {
        _password = baru;
        print('Password berhasil diubah.');
      } else {
        print('Gagal: Password baru minimal 8 karakter.');
      }
    } else {
      print('Gagal: Password lama tidak sesuai.');
    }
  }
}

void main() {
  print('--- Inisialisasi User ---');
  final User user1 = User('admin_alicia', 'securePass987');
  
  print('--- Uji Coba Login ---');
  if (user1.login('admin_alicia', 'securePass987')) {
    print('Login Berhasil!');
  }

  print('\n--- Uji Coba Ganti Password ---');
  user1.gantiPassword('securePass987', 'newSecurePass123');
  user1.gantiPassword('wrongPass', 'anyPass'); // Gagal
}

// Nama   : Alicia Putri Lestari

class RekeningBank {
  String _nomorRekening;
  double _saldo;

  RekeningBank(this._nomorRekening, this._saldo);

  void deposit(double jml) {
    if (jml > 0) {
      _saldo += jml;
      print('Deposit Rp $jml Berhasil.');
    }
  }

  void tarik(double jml) {
    if (jml > 0 && _saldo >= jml) {
      _saldo -= jml;
      print('Tarik Rp $jml Berhasil.');
    } else {
      print('Gagal: Saldo tidak cukup atau input tidak valid.');
    }
  }

  void cekSaldo() {
    print('Rekening: $_nomorRekening | Saldo: Rp $_saldo');
  }
}

void main() {
  var bank = RekeningBank('2514-10002', 3000000);
  bank.cekSaldo();
  bank.tarik(600000);
  bank.cekSaldo();
}
