PENGERTIAN POLIMORFISME DAN ABSTRAKSI

Polimorfisme: Entitas sama dengan perilaku berbeda sesuai konteks. Abstraksi: Sembunyikan detail tak perlu, tampilkan hanya hal esensial.

PEWARISAN

Pewarisan adalah konsep Pemrograman Berorientasi Objek (OOP) yang memungkinkan kelas anak (subclass) untuk mengambil alih serta menggunakan atribut dan fungsi yang dimiliki oleh kelas induk (superclass). Pada bahasa Dart, hubungan pewarisan dibuat dengan menggunakan kata kunci  extends .

// Nama   : Alicia Putri Lestari
// Nim    : 251410011
// Kelas  : SI2A

class Hewan {
  void bersuara() => print('...');
}

class Kucing extends Hewan {
  @override
  void bersuara() => print('Meong');
}

void main() {
  Kucing k = Kucing();
  k.bersuara();
}

PENGANTAR POLIMORFISME

Polimorfisme berarti "banyak bentuk"—konsep inti OOP yang membuat entitas seperti method/kelas memiliki perilaku berbeda sesuai objek/konteksnya. Kode jadi fleksibel, bisa digunakan untuk berbagai objek tanpa logika terpisah, dan bekerja bersama dengan pewarisan serta abstraksi.

POLIMORFISME DENGAN INHERITANCE

Variabel parent bisa simpan objek child berbeda.

// Nama  : Alicia Putri Lestari
// Nim   : 251410011
// Kelas : SI2A

class Bentuk {
  void gambar() => print('Menggambar bentuk');
}

class Lingkaran extends Bentuk {
  @override
  void gambar() => print('Menggambar lingkaran');
}

class Persegi extends Bentuk {
  @override
  void gambar() => print('Menggambar persegi');
}

void main() {
  Bentuk bentuk1 = Lingkaran(); // Polimorfisme
  Bentuk bentuk2 = Persegi();

  bentuk1.gambar(); // Menggambar lingkaran
  bentuk2.gambar(); // Menggambar persegi
}

MANFAAT POLIMORFISME

Kode dinamis, mudah diperluas, bisa digunakan kembali, dan mudah dikelola.

is  mengecek apakah objek termasuk tipe tertentu;  as  melakukan konversi tipe objek agar bisa mengakses method tertentu dengan aman.

ABSTRACT METHOD


Abstract method adalah metode yang hanya dideklarasikan (tanpa badan implementasi) dalam kelas abstrak, dan wajib dibuatkan implementasinya oleh kelas turunannya untuk menjamin konsistensi fungsionalitas antar subclass.

// Nim   : 251410011
// Kelas : SI2A

  void mainkan();
  void info() {
    print('Ini adalah alat musik');
  }
}

class Gitar extends AlatMusik {
  @override
  void mainkan() {
    print('Jreng jreng...');
  }
}

void main() {
  Gitar g = Gitar();
  g.info();
  g.mainkan();
}











INTERFACE DI DART


Interface di Dart adalah kontrak yang mendefinisikan kumpulan metode dan properti yang harus diimplementasikan oleh kelas yang menggunakannya; kelas dapat mengimplementasikan banyak interface untuk memiliki berbagai kemampuan.




// Nama  : Alicia Putri Lestari
// Nim   : 251410011
// Kelas : SI2A

interface class Flyable {
  void terbang() => print('Terbang...');
}

class Bird implements Flyable {
  @override
  void terbang() => print('Burung terbang tinggi');
}

class Plane implements Flyable {
  @override
  void terbang() => print('Pesawat terbang cepat');
}

void main() {
  Bird b = Bird();
  Plane p = Plane();

  b.terbang();
  p.terbang();
}











PERBEDAAN EXTENDS VS IMPLEMENTS
 

- extends: Digunakan untuk pewarisan dari satu kelas induk saja, subclass mewarisi anggota induk dan dapat mengoverride perilakunya.


- implements: Digunakan untuk mengikuti kontrak dari interface atau kelas (bisa banyak), kelas harus mengimplementasikan seluruh metode dan properti yang didefinisikan.








STATIC MEMBERS DAN METHOD


Static Members dan Method di Dart: Properti atau metode yang milik kelas itu sendiri (bukan objek yang dibuat darinya), diakses langsung melalui nama kelas. Static method bisa dijalankan tanpa membuat instance dan tidak dapat mengakses anggota non-statis.




// Nama  : Alicia Putri Lestari 
// Nim   : 251410011
// Kelas : SI2A
class MathUtils {
  static double pi = 3.14;

  static int add(int x, int y) {
    return x + y;
  }
}

void main() {
  print(MathUtils.pi);
  print(MathUtils.add(10, 20));
}

STUDI KASUS: SISTEM BENTUK GEOMETRI


Kelas induk abstrak  Bentuk  sebagai dasar, setiap bentuk (segitiga, persegi panjang) memiliki implementasi berbeda untuk menghitung luas dan keliling, namun bisa digunakan secara seragam




// Nama  : Alicia Putri Lestari
// Nim   : 251410011
// Kelas : SI2A

abstract class Bentuk {
  double hitungLuas();
  double hitungKeliling();

  void info() {
    print('Luas: ${hitungLuas()}, Keliling: ${hitungKeliling()}');
  }
}

class Persegi implements Bentuk {
  double sisi;
  Persegi(this.sisi);

  @override
  double hitungLuas() => sisi * sisi;

  @override
  double hitungKeliling() => 4 * sisi;

  @override
  void info() {
    print('Persegi sisi $sisi → Luas ${hitungLuas()} | Keliling ${hitungKeliling()}');
  }
}

class Lingkaran implements Bentuk {
  double radius;
  Lingkaran(this.radius);

  @override
  double hitungLuas() => 3.14 * radius * radius;

  @override
  double hitungKeliling() => 2 * 3.14 * radius;

  @override
  void info() {
    print('Lingkaran radius $radius → Luas ${hitungLuas()} | Keliling ${hitungKeliling()}');
  }
}

void main() {
  Bentuk b1 = Persegi(4);
  Bentuk b2 = Lingkaran(7);

  b1.info();
  b2.info();
}

CLASS 3: CLASS MATHUTILS


Kelas dengan fungsi statis matematika, diakses langsung tanpa instance sehingga praktis untuk digunakan berulang.




// Nama  : Alicia Putri Lestari
// Nim   : 251410011
// Kelas : SI2A

import 'dart:math';
class MathUtils {
  static final Random _random = Random();

  static int faktorial(int n) {
    if (n < 0) {
      throw ArgumentError('n tidak boleh negatif');
    }
    if (n <= 1) return 1;
    return n * faktorial(n - 1);
  }

  static bool isPrima(int n) {
    if (n <= 1) return false;
    for (int i = 2; i <= sqrt(n).toInt(); i++) {
      if (n % i == 0) return false;
    }
    return true;
  }

  static double bulatkan(double angka, int desimal) {
    double faktor = pow(10, desimal).toDouble();
    return (angka * faktor).round() / faktor;
  }

  static int randomInt(int min, int max) {
    if (min > max) {
      throw ArgumentError('min tidak boleh lebih besar dari max');
    }
    return min + _random.nextInt(max - min + 1);
  }

  static double randomDouble(double min, double max) {
    if (min > max) {
      throw ArgumentError('min tidak boleh lebih besar dari max');
    }
    return min + _random.nextDouble() * (max - min);
  }

  static double rataRata(List data) {
    if (data.isEmpty) {
      throw ArgumentError('List tidak boleh kosong');
    }
    double total = data.reduce((a, b) => a + b);
    return total / data.length;
  }

  static double median(List data) {
    if (data.isEmpty) {
      throw ArgumentError('List tidak boleh kosong');
    }

    List sorted = List.from(data)..sort();
    int n = sorted.length;

    if (n % 2 == 1) {
      return sorted[n ~/ 2];
    } else {
      return (sorted[n ~/ 2 - 1] + sorted[n ~/ 2]) / 2;
    }
  }
}

void main() {
  print(MathUtils.faktorial(5));
  print(MathUtils.isPrima(7));
  print(MathUtils.bulatkan(3.14159, 2));
  print(MathUtils.randomInt(1, 10));
  print(MathUtils.randomDouble(1.0, 5.0));
  print(MathUtils.rataRata([1, 2, 3, 4, 5]));
  print
}











CHALLENGE


- Sistem Media Player: Buat sistem dengan kelas abstrak dan interface;  AudioPlayer  dan  VideoPlayer  terapkan metode sama tapi aksi berbeda sesuai media, latih abstraksi dan polimorfisme.




// Nama  : Alicia Putri Lestari
// Nim   : 251410011
// Kelas : SI2A

abstract class MediaPlayer {
  void play();
}

abstract class Playable {
  void play();
}

abstract class Pausable {
  void pause();
}

class AudioPlayer extends MediaPlayer implements Playable, Pausable {
  @override
  void play() {
    print('Memutar audio');
  }

  @override
  void pause() {
    print('Pause audio');
  }
}

class VideoPlayer extends MediaPlayer implements Playable, Pausable {
  @override
  void play() {
    print('Memutar video');
  }

  @override
  void pause() {
    print('Pause video');
  }
}

void main() {
  AudioPlayer a = AudioPlayer();
  VideoPlayer v = VideoPlayer();

  a.play();
  a.pause();

  v.play();
  v.pause();
}
